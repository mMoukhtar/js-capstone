# Welcome to CAPSTONE you ultimate travel planner app

## Getting Started

####git clone https://github.com/mMoukhtar/js-capstone.git
####npm install && npm start
####add .env
####port=8000
####geonames_api_key=

### Start App in Dev mode

npm run start:dev
this will run goole chrome on port: 9000,

### Build App

npm run build

### Start the Server

npm run start
